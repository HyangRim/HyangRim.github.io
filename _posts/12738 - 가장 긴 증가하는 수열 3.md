# 12738 - 가장 긴 증가하는 수열 3

들어오는 인풋 N의 크기가 1000000이라서 일반적인 방법으로는 시간초과가 나게 됨. 

그래서 LIS를 사용할 것. 

1. Arr에 들어가는 값을 LIS라는 배열에 삽입할 것. 
2. LIS배열은 증가하는 부분 수열 규칙에 어긋나지 않게, LIS배열의 적절한 위치를 찾으면서 값을 재설정하거나, 값을 넣어줄 것. 

예제에 나온대로 입력이 

```
6
10 20 10 30 20 50
```

일 때. 

처음에 10을 넣으면. 어차피 비교할 것은 하나도 없으니까. 일단 

arr이라는 배열에 0번째로 들어감. 그 후에 LIS 배열에 넣는데 위에서 말했던 대로 비교할 것도 뭐도 아무것도 없으니까 LIS배열의 첫 번째에 넣어주면 된다.

LIS[] = {10(1)}임. (1)은 arr[1]의 10이라는 의미다. 

두 번째로 20을 넣자. 

arr[2]에 들어가고, 문제의 규칙. 오름 차 순으로 가장 긴 수열을 찾는거니까, 20은 10보다 크고 다른 비교할 게 없다. 그러니까. 그냥 10 뒤에 넣어주면 되니까. 

LIS[] = {10(1), 20(2)}

세 번째로 10을 넣는데, 여기서부터 생각을 할 게 생긴다.

LIS에 '10'을 넣는데, 규칙은 가장 긴 증가하는 수열을 찾는거니까. LIS 세 번째. 즉, LIS[] = {10,20,10}은 규칙에 어긋난다. 그래서 '10'은 저 20보다는 일찍 있어야한다. 그러니까 '10(3)'의 위치는 LIS의 1번째 위치(10)밖에 없다. 

그러면 LIS[] = {10(3), 20(2)}가 된다. 

네 번째로 30이 들어오는데, 30은 생각할 것도 없이 10, 20이후에 들어가면 되는 거기 때문에 LIS[] = {10(3), 20(2), 30(4)}가 된다.

다섯 번째로 20이 온다. 요것도 세 번째랑 마찬가지로, 20이 올 것은 현재로서는 10과 30의 사이. 즉, 20의 자리밖에 없다. 그래서 

LIS[] = {10(3), 20(5), 30(4)}가 된다.

마지막으로 오는 50을 보자.

50도 별로 생각할 것도 없이 10, 20, 30의 뒤에 오면 되니까

LIS[] = {10(3), 20(5),30(4), 50(6)}이 LIS의 최종 완성본이다. 

근데 이게 들어가는 input의 크기가 1000000이라서, N*N이 나온다. 그래서 시간초과가 뜰텐데, 이를 해결하기 위해서 이분 탐색을 사용한다. 

```cpp
#include <iostream>
#include <vector>

int N;
int arr[1000001];
int index[1000001];

using namespace std;

vector<int> v1;
vector<int> ans;
int main() {
	ios::sync_with_stdio(false);
	cin.tie(0);
	cout.tie(0);
	cin >> N;
	for (int x = 1; x <= N; x++) {
		cin >> arr[x];
	}

	for (int x = 1; x <= N; x++) {
		if (!v1.size()|| v1[v1.size() - 1] < arr[x])
		{//만약에 처음 들어가거나, 지금까지 들어온 것 중에 가장 클 경우에
			v1.push_back(arr[x]);// 그냥 넣으면 된다.
			index[x] = v1.size() - 1;//index위치도 그냥 그대로 넣으면 된다.
		}
		else// 만약 10,30중에 20이 들어온거로 가정해서 위치를 찾아야할 경우에.
		{
			int left = 0;
			int right = v1.size() - 1, mid;
			while (left < right)
			{
				mid = (left + right) / 2;

				if (v1[mid] >= arr[x]) right = mid;//이분 탐색.
				else left = mid + 1;
			}//이분 탐색해서 얻은 위치. 

			v1[left] = arr[x];// 이분 탐색으로 30자리에다가 20넣기..
			index[x] = left;//위치를 인덱스에 넣기. 
		}
	}

	cout << v1.size() << "\n";// 사이즈가 가장 긴 증가하는 수열.
	return 0;
}
```
